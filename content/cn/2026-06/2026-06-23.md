## 政策合规 🏛️

- 🇬🇧 **Scattered Spider 黑客开庭首日认罪** — 两名黑客在英国法庭对2024年8月导致伦敦交通局（TfL）瘫痪的网络攻击认罪。该案涉及大规模账户窃取与数据勒索，或推动英国立法机构加快出台针对勒索软件强制报告的新规。 [阅读原文](https://krebsonsecurity.com/2026/06/scattered-spider-hackers-plead-guilty-on-day-1-of-trial/)

- 🔧 **GitHub 更新 actions/checkout 阻断 Pwn Request 攻击** — GitHub 官方加固 actions/checkout 组件，默认拦截利用 `pull_request_target` 触发的供应链投毒模式。此举被视为平台层对 CI/CD 安全合规的补位，国内 DevOps 团队应同步检查工作流配置。 [阅读原文](https://thehackernews.com/2026/06/github-updates-actionscheckout-to-block.html)

- 🕵️ **Tata Electronics 确认遭攻击，数据已被黑客泄露** — 印度塔塔集团旗下电子制造子公司承认IT系统受损，黑客已公开部分数据。虽然事件发生在国外，但对于拥有全球供应链的国内代工企业而言，是一次重要的外部威胁预警。 [阅读原文](https://www.bleepingcomputer.com/news/security/tata-electronics-confirms-cyberattack-as-hackers-leak-data/)

## 漏洞威胁 🚨

- ⚠️ **Cisco Unified CM 漏洞 CVE-2026-20230 已遭在野利用** — 该高危SSRF漏洞影响 Cisco Unified Communications Manager Server，攻击者可利用它绕过访问控制。Cisco已发布补丁，建议国内呼叫中心与统一通信用户立即更新。 [阅读原文](https://www.bleepingcomputer.com/news/security/cisco-unified-cm-sme-flaw-cve-2026-20230-now-exploited-in-attacks/)

- 🩺 **医疗科技公司 Xolis 数据泄露影响140万人** — 攻击者通过钓鱼邮件入侵网络，窃取了包含健康信息在内的敏感数据。医疗数据在黑市价格高昂，国内医疗机构应警惕类似钓鱼+勒索的组合攻击手法。 [阅读原文](https://www.bleepingcomputer.com/news/security/healthtech-firm-xolis-suffers-data-breach-impacting-14-million-people/)

- 📦 **恶意 npm 包冒充 PostCSS 工具，投递 Windows 远控木马** — 安全厂商发现一批伪装成 PostCSS 插件的 npm 包，安装后会下载 RAT。国内 Node.js 开发者应验证依赖来源，警惕名称仿冒的“抢注”包。 [阅读原文](https://thehackernews.com/2026/06/malicious-npm-packages-pose-as-postcss.html)

## API 安全 🔑

- 🔓 **SAML 认证绕过新手法：属性污染与命名空间混淆** — PortSwigger 研究员公布了在 Ruby 和 PHP SAML 实现中实现全认证绕过的技术细节，利用解析器级不一致性。SAML 仍广泛用于企业 SSO，国内集成商需评估自身库是否受影响。 [阅读原文](https://portswigger.net/research/the-fragile-lock)

- 🤖 **Dify 平台爆 DifyTap 漏洞，AI 聊天可跨租户窃取** — 安全研究者披露开源 AI 工作流平台 Dify 的4个漏洞，允许攻击者跨租户访问对话历史与模型输出。Dify 在国内有大量企业用户，建议立即升级版本。 [阅读原文](https://thehackernews.com/2026/06/researchers-detail-difytap-flaws-in.html)

- 🏆 **2025年度 Top 10 Web 黑客技术投票进行中** — PortSwigger 发起第19届社区评选，涵盖大量 API 绕过与 OAuth 滥用手法。国内安全团队可借此榜单及时了解全球攻击趋势，调整 API 防护策略。 [阅读原文](https://portswigger.net/research/top-10-web-hacking-techniques-of-2025-nominations-open)

- 📊 **2025年度 Top 10 Web 黑客技术榜单揭晓** — 社区票选年度最具创新性 Web 安全研究，涵盖请求走私、缓存投毒、OAuth 链式攻击等。这是安全从业者不可错过的年度技术风向标。 [阅读原文](https://portswigger.net/research/top-10-web-hacking-techniques-of-2025)

- 🐙 **GitHub 加固 actions/checkout：开源 CI/CD 安全里程碑** — 通过默认阻塞 pwn request 攻击模式，actions/checkout 成为更安全的代码签出工具。国内基于 GitHub Actions 的流水线可自动受益，建议同步阅读官方变更日志。 [阅读原文](https://thehackernews.com/2026/06/github-updates-actionscheckout-to-block.html)

- 🚫 **npm 生态再遭投毒：PostCSS 仿冒包来袭** — 除了直接下载远控木马，这些包还可能窃取环境变量与令牌。使用 `npm audit` 并开启软件来源签名验证，是阻断此类攻击的有效措施。 [阅读原文](https://thehackernews.com/2026/06/malicious-npm-packages-pose-as-postcss.html)
