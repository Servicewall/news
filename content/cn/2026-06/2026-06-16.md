📅 2026/6/16 安全日报 · 数据与API安全

## 1️⃣ 政策合规

🇨🇳 **公安部通报37款违规应用，电商类超七成，小程序纳入监管**  
依据《网络安全法》《个人信息保护法》，公安部检测发现37款移动应用存在未公开收集规则、超范围采集等违规行为，其中电商类占比超70%，支付宝/微信小程序成重灾区。监管部门明确：小程序不再是“法外之地”。  
🔗 [嘶吼 RoarTalk](https://www.4hou.com/posts/PGXA)

🌍 **CISA 紧急警告：cPanel 插件漏洞 CVE-2026-54420 已被利用**  
美国网络安全与基础设施安全局（CISA）要求联邦机构在3天内修复该漏洞，攻击者可通过未认证方式远程执行代码。该漏洞影响多个cPanel环境，建议企业立即检查并更新。  
🔗 [BleepingComputer](https://www.bleepingcomputer.com/news/security/cisa-warns-of-another-actively-exploited-cpanel-plugin-flaw/)

🌍 **微软6月Patch Tuesday创纪录：修复近200个漏洞**  
微软发布2026年6月安全更新，修补了Windows及支持软件中的近200个安全漏洞，创历史新高。其中包括多个已被公开利用的零日漏洞，建议用户尽快安装补丁。  
🔗 [Krebs on Security](https://krebsonsecurity.com/2026/06/a-record-breaking-patch-tuesday-for-june-2026/)

## 2️⃣ 漏洞威胁

🇨🇳 **梆梆安全Q1报告：超八成APP存隐私违规，数据境外外发风险突出**  
报告显示，2026年一季度国内移动应用隐私违规率高达82%，数据向境外传输的现象显著增加。盗版仿冒APP、高危漏洞和第三方SDK滥用是主要风险源，企业需加强移动应用安全审计。  
🔗 [嘶吼 RoarTalk](https://www.4hou.com/posts/pKwQ)

🌍 **攻击者利用三个 Fortinet FortiSandbox 漏洞，其中1个刚刚修复**  
Defused Cyber 报告称，攻击者正在同时利用三个 FortiSandbox 漏洞进行入侵，其中包括上周才被修补的高危漏洞。建议用户立即升级至最新版本，并检查是否有异常访问日志。  
🔗 [The Hacker News](https://thehackernews.com/2026/06/attackers-exploit-three-fortinet.html)

🌍 **勒索软件 DragonForce 利用 Microsoft Teams 中继隐藏恶意流量**  
DragonForce 使用名为“Backdoor.Turn”的自定义恶意软件，将命令与控制通信伪装成正常的 Teams 回会话流量，从而绕过防火墙检测。该手法表明攻击者正越来越多地滥用合法云服务。  
🔗 [BleepingComputer](https://www.bleepingcomputer.com/news/security/ransomware-gang-abuses-microsoft-teams-relays-to-hide-malicious-traffic/)

## 3️⃣ API 安全

🌍 **Google Vertex AI SDK 缺陷：攻击者可通过 Bucket 劫持接管模型上传**  
Python SDK 中存在存储桶命名空间冲突漏洞，未授权攻击者若能预测受害者使用的 GCS 存储桶名称，即可在模型上传时注入恶意代码，在 Google 基础设施内执行任意操作。  
🔗 [The Hacker News](https://thehackernews.com/2026/06/google-vertex-ai-sdk-flaw-let-attackers.html)

🌍 **PortSwigger 研究：SAML 认证的新型绕过技术**  
研究人员披露 Ruby 和 PHP 生态中 SAML 解析器的多种绕过手法，包括属性污染、命名空间混淆和解析器不一致，可导致完全认证绕过。涉及多个开源库，企业需更新实现逻辑。  
🔗 [PortSwigger Research](https://portswigger.net/research/the-fragile-lock)

🌍 **2025年度十大Web黑客技术揭晓：API攻击手法占主流**  
PortSwigger 发布第19届社区投票结果，入选技术包括 OAuth 混淆代理、GraphQL 注入、JWT 算法混淆等，凸显 API 安全攻防的持续演进。安全团队应关注这些实战技术。  
🔗 [PortSwigger Research](https://portswigger.net/research/top-10-web-hacking-techniques-of-2025)

## 5️⃣ 开源工具

🌍 **恶意 JetBrains Marketplace 插件窃取开发者 AI API 密钥**  
至少15个在 JetBrains 插件市场上架的恶意插件被用于窃取 OpenAI、Anthropic 等 AI 服务的 API Key。攻击者通过仿冒知名插件诱导下载，建议开发者审查已安装插件来源。  
🔗 [BleepingComputer](https://www.bleepingcomputer.com/news/security/malicious-jetbrains-marketplace-plugins-steal-ai-api-keys-from-developers/)

🌍 **SAML 认证库漏洞分析：ruby-saml/php-saml 存在解析器差异**  
PortSwigger 公开了多款流行开源 SAML 库中的安全短板，攻击者可利用 XML 解析器差异绕过身份验证。建议使用相关库的项目及时更新到修复版本或应用缓解措施。  
🔗 [PortSwigger Research](https://portswigger.net/research/the-fragile-lock)
