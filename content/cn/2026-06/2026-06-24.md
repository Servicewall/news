## 1. 政策合规 ⚖️

**CISA 警告：Lantronix EDS5000 高危漏洞正被积极利用**  
美国网络安全与基础设施安全局（CISA）周二发出警报，通报 Lantronix EDS5000 系列设备存在一个正在被积极利用的关键安全漏洞，要求联邦机构立即采取缓解措施。该漏洞若被成功利用，可能导致设备完全失控，提醒相关企业尽快排查并打补丁。  
🔗 https://thehackernews.com/2026/06/cisa-warns-critical-lantronix-eds5000.html

**Scattered Spider 黑客在审判首日认罪**  
两名男子本周在英国法庭上对 2024 年 8 月导致伦敦交通局（TfL）系统瘫痪的网络攻击表示认罪。该攻击曾造成公共交通服务大面积中断，两名嫌疑人面临多项刑事指控。此次认罪标志着针对高调勒索团伙 Scattered Spider 的司法调查取得关键进展。  
🔗 https://krebsonsecurity.com/2026/06/scattered-spider-hackers-plead-guilty-on-day-1-of-trial/

**DraftKings 黑客 "Snoopy" 被判入狱 18 个月**  
一名化名 "Snoopy" 的 21 岁黑客因参与 2022 年 11 月的 DraftKings 账户攻击事件，被判处 18 个月监禁。该攻击导致大量用户账户被盗，黑客通过撞库等手段获取访问权限并实施提现。法院判决彰显了针对体育博彩平台网络犯罪的零容忍态度。  
🔗 https://www.bleepingcomputer.com/news/security/draftkings-hacker-snoopy-sentenced-to-18-months-in-prison/

## 2. 漏洞威胁 🛡️

**Amadey 和 StealC 恶意软件网络被摧毁，回收 2700 万条凭证**  
由执法部门联合 Bitdefender、Bitsight、ESET、微软等私营企业共同发起的协调行动，成功捣毁了支撑 Amadey 与 StealC 恶意软件运营的犯罪基础设施。行动中回收了超过 2700 万条被盗的登录凭证，极大遏制了针对个人和企业的凭证窃取活动。  
🔗 https://thehackernews.com/2026/06/amadey-and-stealc-malware-network.html

**Cordyceps CI/CD 漏洞曝光：超 300 个 GitHub 仓库面临供应链攻击风险**  
安全研究人员揭示了一类新的 CI/CD 工作流弱点模式，攻击者可利用该模式劫持工作流并破坏开源软件供应链。该漏洞（代号 Cordyceps）影响大量 GitHub 仓库，使得攻击者能在构建阶段注入恶意代码，对下游用户构成严重威胁。  
🔗 https://thehackernews.com/2026/06/cordyceps-cicd-flaws-expose-300-github.html

**恶意 Edge 扩展 “Edgecution” 利用原生消息桥接绕过沙箱部署后门**  
一种名为 “Edgecution” 的恶意 Microsoft Edge 扩展被发现在勒索软件攻击中用于逃逸浏览器沙箱，进而部署 Python 后门。该扩展滥用原生消息（Native Messaging）机制作为桥梁，成功绕过浏览器安全限制，实现持久化驻留和远程控制。  
🔗 https://www.bleepingcomputer.com/news/security/malicious-edge-extension-abuses-native-messaging-as-bridge-to-malware/

**Mandiant 披露 Cisco SD-WAN 零日攻击细节：攻击者如何获取 root 权限**  
安全公司 Mandiant 详细披露了黑客如何利用 Cisco Catalyst SD-WAN 漏洞（编号 CVE-2026-20245）在零日攻击中创建恶意 root 账户。攻击者通过精心构造的请求，突破了设备的身份验证机制，获得了对目标设备的完全控制权。  
🔗 https://www.bleepingcomputer.com/news/security/mandiant-reveals-how-cisco-sd-wan-zero-day-attacks-gained-root-access/

## 3. API 安全 🔑

**SAML 认证新型绕过：属性污染与命名空间混淆**  
PortSwigger 研究团队揭示了一系列针对 Ruby 和 PHP 生态中 SAML 认证实现的全新绕过技术。通过利用解析器级别的不一致性（包括属性污染、命名空间混淆、XML 注释滥用等），攻击者无需有效凭证即可完成认证绕过，严重威胁使用 SAML 的 API 网关和单点登录系统。  
🔗 https://portswigger.net/research/the-fragile-lock

**2025 年度十大 Web 黑客技术提名开放，API 攻击手法成焦点**  
PortSwigger 发起的第 19 届“年度十大 Web 黑客技术”评选已进入投票阶段。今年提名涉及大量新颖的 API 攻击手法，包括 GraphQL 注入、OAuth 重定向滥用等。社区投票结果将直接反映过去一年 Web 安全研究中最具创新性的技术方向。  
🔗 https://portswigger.net/research/top-10-web-hacking-techniques-of-2025

## 4. 行业动向 📈

**“Popa” 僵尸网络与以色列上市公司关联：四年来强制数百万电视盒转发流量**  
安全调查发现，一个名为 “Popa” 的庞大 Android 僵尸网络在过去四年间控制了数百万台消费级电视盒，强制其转发互联网流量以实施广告欺诈、账户接管和大规模数据窃取。该僵尸网络背后指向一家公开上市的以色列公司。  
🔗 https://krebsonsecurity.com/2026/06/popa-botnet-linked-to-publicly-traded-israeli-firm/

**勒索软件组织 “The Gentlemen” 快速崛起：揭秘其招募策略**  
一个名为 “The Gentlemen” 的网络犯罪集团已成为按受害者数量计算的第二活跃的勒索软件团伙。该团伙通过激进的招募策略吸引了大量有才华的黑客，并在短时间内建立起高效的攻击链。Krebs on Security 深入调查了该组织的运作模式和核心成员身份。  
🔗 https://krebsonsecurity.com/2026/06/who-runs-the-ransomware-group-the-gentlemen/
